# Math Basic Notes

## 牌效策略

基于 **向听数** 与 **进张数**, 适当考虑 **高番数**:

- 摸入后可以减少向听数的牌称为**进张**.
- 摸入后可以增加进张数的牌称为**改良**.
- **进张**与**改良**合称为**有效牌**.

### 单张牌效

切牌顺序:

- `14`/`69` 中的 `1` 与 `9`.
- 客风牌: 优先切上家门风, 上家鸣牌后可多一次摸切机会.
- `1` `9`: 孤立的老头牌比任何一种对子和搭子的价值都要低.
- 役牌: 先场风/三元牌, 后自风牌, 最后双风牌 (两番).
- `2` `8`.
- `3` `4` `5` `6` `7`.
- 搭子进相同张直接模切, 成型面子进行空切:
  - 使对手误以为成型面子周围危险.
  - 重置对手安全牌判断.
- 早巡对手捨牌的**周边牌进张较为容易**:
  - 进张可能: 大概率不在对手手牌, 而在**牌山中**.
  - 鸣牌可能: 对手后续摸到早巡捨牌的周边牌 (孤张废张), 大概率会直接摸切.
  - 听牌可能: 对手放铳率增加.
- 连筋孤张尽早打出，否则进张切牌时容易放铳.
- 搭子**足够** + 打点**足够** (不需要宝牌/三色靠张) 时,
  **优先切高危**险度孤张 (宝牌/中张靠张/生张字牌).

### 对子牌效

#### 对子基本牌效

- **早巡**不应固定雀头, 应该把雀头拆开和其他牌组成搭子, **中晚巡**自然进张形成雀头或切牌固定雀头.
- 役牌为**唯一雀头**时, 若牌型较好(一向听), 且立直门清打点显著高于役牌打点, 则不急于鸣役牌.
- 客风对子:
  - 攻可作雀头听平和, 守可作安牌防一发.
  - 若对子或搭子超载, 可以优先切掉作断幺九手役.
- 非现物中张对子进张十分困难, 有极大概率**与对手死锁**.

#### 两对子牌效

- 两对子存在双碰进张, 是快速听牌的诀窍.

#### 三对子牌效

- 三对子相比于两对子, 仅增加 2 张刻子进张.
- 有役时 (役牌, 断幺九, 对对和):
  - 考虑保留三组对子, 可将**碰**作为进张手段, 三对子可作为**快速听牌**手段.
- 两向听时:
  - **一向听顶峰理论**: 最大化**一向听进张**数, 在两向听时可考虑保留三组对子, 切掉较弱三牌复合型.
- 门清时 (立直或平和):
  - 考虑切掉一组对子 (同时保证不退向): 切两面对 -> 切嵌张对 -> 切边张对.
  - 考虑**向内靠拢**, 切幺九对.

#### 四对子牌效

四对子: 进张较差时, 可考虑七对子手役.

### 搭子牌效

#### 搭子基本牌效

- 搭子强度: 两面 > 连嵌 > 中张嵌张 >> `24`/`68`嵌 > `13`/`79`嵌 > 边张.
- 双碰进张优于连嵌进张, 特别是在有役牌时, **碰**的几率远大于**吃**.
- **高向听**时, 禁止切掉唯一一组雀头 (切掉后会导致退向).
- **奇数**枚有利: 3/5/7 张同色牌比偶数枚更优.
- 四连型, 中膨型, 亚两面型混合时, **切掉亚两面型**, 高概率是正解.
- 多种复合型混合成复杂形状时, **保留复杂形状, 切牌固定两面搭子或完整面子**, 高概率是正解.
- 中巡好型搭子足够时, 可考虑**保留安全牌**进行防守, 亦可用安全牌进行追立:
  完全一向听时, 上张熟张字牌, 可以考虑切掉两面对中的对子补强牌 (仅损失 2 枚非平和进张),
  保留熟张字牌 (**安全牌**), 切掉**危险两面对** (牌河较少同色牌) 中的对子补强牌,
  攻可切字牌安全立直, 守可作安牌防止一发.

**有效牌重复**效应:

- 筋牌: `147`, `258`, `369` 同一筋上存在有效牌重复效应.
- 边张: `14` / `69` 存在时, `1` `9` 价值甚至低于客风牌.
- 存在二度进张的边张搭子, 切掉边张搭子保留浮牌比较有利.
- `14` 进 `2`/`3`, 形成 `124`/`134`, 仍然切 `1`, 所以可以在 `14` 时就切 `1`.

#### 五搭子理论

**五搭子理论** - 五搭子饱和, 六搭子不利:

- 五搭子型基本上只要切掉价值最低的浮牌.
- 搭子超载时, 优先切掉弱搭子:
  - 强搭子: 两面搭, 连嵌搭, **宝牌搭**, 役牌搭, **三色搭**, 染手搭, 三牌复合搭.
  - 弱搭子: 嵌张搭, 边张搭, 损失手役牌 (幺九牌), **绝张搭** (e.g 损失四枚 `2` 的 `34` 两面搭).
  - 搭子超载全为好型时, 优先**切掉浮张**, 然后切掉超载搭子.
  - 搭子超载存在愚型时, 优先切掉最弱搭子, 暂时**保留中张浮张**, 后续形成中张好型搭子后, 继续切掉最弱搭子, 直至剩下至多五组搭子.
  - **低效率二度进张**: 考虑有效牌重复效应, 可以切掉同色超载搭子 (进张数损失最小).
    如 `1245` 二度进张 `3`, `3467` 二度进张 `5`.
  - 存在复合型时, 可以先切边张: 价值低.
  - 不存在复合型时, 可以先切中张: 危险度高.

### 三牌复合型

- 两面对: `233`.
- 嵌张对: `244`.
- 边张对: `122`.
- 连嵌张: `246`.
  - 两面嵌张: `2` + `456` + `67` -> `246` + `567`.
  - 长连嵌张: `134568`, 进 `2` 或 `7` 都可形成两面子.
  - 弱于两面搭子: 需要 3 张牌, 听牌型为嵌张听.

搭子进张成为搭子 + 对子复合型, 称该进张为**补强牌**:

- 两对子双碰:
  当出现两组复合型二择时,
  切掉**搭子部位较弱**的复合型的**搭子**,
  固定两对子双碰牌型.
- 三对子不利:
  当出现三对子不利时,
  切掉**搭子部位较强**的复合型的**对子** (e.g 两面对),
  **让补强牌辅助较弱的搭子**,
  固定强搭子牌型 (复合型 - 对子 = 搭子),
  即**固定好型**,
  切掉三对子不利牌型.

`13345667m` + `123788s`:

先根据五块理论与中膨复合型, 将手牌分为 `133m` + `456m` + `67m` + `123s` + `788s`.
可以此时已经形成了有效的五块搭子, 需要切掉两个三牌复合型中的一个, 保持张数的正收益.
对于`对子` + `搭子`的复合型, 可以采取固定好型的策略, 切掉 `8s`, 固定 `78s` 两面型.
显然切掉 `3m` 去固定 `13m` 嵌张型是低效率的切法.

### 四牌复合型

#### 四连型

又称延伸单骑, 具有良好的**延展性**:

- 良型: `3456` -> `34` + `56`, 常常将其视为**两面 + 两面**, 而不是单骑进张.
- 次良型: `1234` -> `123` + `4`, 常常将其视为单骑进张, 而不是**边张 + 两面**.
- 缺少对子时, 四连型的延伸单骑价值上升; 不缺对子时, 四连型的延伸单骑价值下降.

#### 中膨型

- 良型: `3445` -> `34` + `45`, 两面 + 一杯口, **好型进张 + 一杯口进张**.
- **恶型**: `1223` -> `123` + `2`, 价值较低, 接近于单张 `2`, 不存在连续型时, 可优先切掉 `2`.

#### 亚两面型

- `3345` -> `34` + `35` / `33` + `45` / `3` + `345`.
- 缺少对子时, `3345` 有一定价值, 稍稍高于单张 `3`.
- 不缺对子时, `3345` 价值较低, 接近于单张 `3`, 可优先切掉 `3`, 保留其他复合型.
- **恶型**: `1123` 价值接近于单张 `1`, `7899` 价值接近于单张 `9`.
- 总体来说, 亚两面型作两面能力**与孤张无太大区别**.
- `3345` -> `34` + `35`, 可以通过鸣牌改良成 `34` 两面.
- 亚两面型可以直接用于听牌, 存在转二杯口可能性, 可以形成 `33345` 暗刻复合型.

#### 跳一型

- `3567` -> `35` + `67`.
- `1235`/`5789` 接近于孤张 `5`.
- `1345`/`5679` 中的 `1`/`9` 比孤张 `1`/`2`/`8`/`9` **强**.
- `5679`: `9` **不再**受制于重复牌效应, 进 `8` 可以形成五连型好型, 形成 `56` 两面搭.
- `3457` 可以形成良型 `34567`, 利用**吃**可以形成两面子.

#### 二度进张复合型

- `3556` -> `35` + `56`: 存在二度进张 `4`, 当搭子超载时, 优先切掉 `3`.
- `3467`: 嵌张二度进张.
- `3344`: 完全二度进张, 只有存在其他复合型 (一杯口) 或双碰听时有价值.
- **恶型**`6889`: 边张二度进张, `69` 连筋恶型, 当搭子超载时, 优先切掉 `9`.

#### 四牌复合型强度

- 四牌复合型**强度**: 四连型 > 中膨型 >> 亚两面型 > 跳一型 > 二度进张恶型.

`1234p` 与 `1123s`:

当手牌缺少对子时, 四连型中的 `1p` 可以解放 `4p`, 进张下一个 `4p` 时, 可以形成面子 + 雀头,
即 `1p` + `234p` -> `123p` + `4p`, 使得 `1p` 价值约等于 `4p`.

而亚两面型中的 `1s` + `123s` 却无法解放 `3s`, 使得 `1s` 价值约等于 `1s`.
比较 `4p` 与 `1s` 价值, 可以得出此时应该切 `1s`, 保留 `1p`.

### 多张复合型

- 五连型: 进张超好型, 且容易形成雀头.
- 暗刻复合型: `333` + `45` -> `33` + `345`, 缺少对子时, 价值上升.
- 暗刻跳一型: `333` + `5`, **暗刻**旁边的**浮牌不适合靠张听牌**.
- 两顺子型: `344556`: `345` + `456`.
- 两面一杯口型: `34455` -> `345` + `45`.
- 两面中嵌型: `23578` 切 `5` (有效牌重复).
- 两面两嵌型: `455679` -> `456` + `579`, 进张 `368`.
- 宝牌连续型: `11234` (Dora `4`), 用筋牌 `1` 作雀头, 摸入 `4` 时可切 `1` 极具迷惑性.
- 跳对子型: `335577`, **跳对子型切中间对子**, 一般选择切 `5`, 固定中间**连嵌好型**, 保留**边张雀头**可能.
- **好型** 八枚翼对称型 (两组中膨型): `34456778` -> 顺子 + 两嵌型 + 对子, 保留此复合型.
- **好型** 九枚翼对称型 (两组中膨型 + 中张): `233456778`, 两组顺子 + 两嵌型, 进张 `1469`, 保留此复合型.
- `23` + `345` + `78`: 切 `8`, 不损失进张.
- `2267888`: `22` + `88` + `678` / `67` + `22` + `888`.

## 向听策略

### 两向听策略

- `面子` + `雀头` + 3 `搭子` + 孤张: 优先切掉低价值孤张.
- `面子` + 3 `对子`搭 + 孤张: 切掉其中一组对子, 固定好型, 留下孤张进张可能.
- `面子` + `雀头` + 4 `搭子` (无孤张):
  - 存在愚型搭子时, 优先切掉最弱搭子.
  - 不存在愚型搭子时, 可以切掉两面对补强牌固定两面好型.

### 一向听策略

- 接近听牌时 (不再缺少搭子),
  可考虑切掉四牌复合型 (四张牌挤占空间) 退回面子 (固定好型),
  保留三牌复合型 (两面对/嵌张对/连嵌张),
  固定 `搭子` + `搭子` 型,
  可以提升直接进张数量.
- 只有 1-2 张进张数差别时, 优先考虑改良和听牌型.
- 早巡时可以考虑**高打点**进行退向:
  - 断幺九/平和: 切边张搭子/对子, 追求断幺九或平和好型.
  - 染手: 切同色搭子 (甚至面子).
  - 进行改良时 (无法改变打点) **禁止退向**.
- **完全一向听**:
  - 2 `面子` + `雀头` + `两面` + `两面对`.
  - 6 种 20 枚进张 (4 种 16 枚两面进张 + 2 种 4 枚双碰进张).
  - 当双碰进张数不理想时 (少于 4 枚), 则**保留安全牌**, 切掉两面对补强牌.
- **无头一向听**:
  - `暗刻` + 2 `面子` + 2 `搭子` 一向听, 不固定雀头进张数更多 (拆开雀头形成面子与搭子).
  - 3 `面子` + 2 `搭子` 一向听, 可能形成单骑听牌.
  - 3 `面子` + 1 `搭子` 一向听, 优先考虑改良和听牌型.
- **撞头一向听** (**靠张一向听**):
  - 3 `面子` + `雀头` + 2 `单张`: e.g `1223455568p7s555m` 切 `2p` 成为撞头一向听 (`8p` `7s` 靠张).
  - 质量较高的一向听, 单张与面子形成复合型后会进一步提升进张数量.

### 听牌策略

#### 基本听牌形式

- 两面听: `23` 听 `1` `4`.
- 双碰听: `22` `55` 听 `2` `5`.
- 嵌张听: `24` 听 `3`.
- 边张听: `12` 听 `3`, `89` 听 `7`.
- 单骑听
  - 雀头.
  - 和牌率: 熟张字牌 > 幺九牌 > (外) 壁引卦 > (边) 筋引卦.
- 听牌数: 两面听 > 双碰听 > 嵌张听 > 边张听 > 单骑听.
  - 两面听优势明显, 听牌数为 8.
  - 单骑听变化多, 难以防守, 特别是听熟张字牌容易荣和.

#### 复合听牌形式

- 四连型与亚两面型听牌数都为 6.
- 暗刻复合型:
  - `1222`: 边张单骑, `1` + `222` / `12` + `22`, 听牌数为 7.
  - `3555`: 嵌张单骑, `3` + `555` / `35` + `55`, 听牌数为 7.
  - `3444`: 两面单骑, `3` + `444` / `34` + `44`, 听牌数为 11.
- 三面听:
  - 听三种牌.
  - 基本三面听: `34567` -> `34` + `567` / `345` + `67`, 听牌数为 11.
  - 单骑三面听: `1234567`, `2234567`, `4566789`, 听牌数为 9.
  - 混合三面听: `23445566`, 可以形成一杯口.
  - 多面听时, 三暗刻也可看作`雀头`+`单张`.

#### 听牌技巧

- 满贯以下听高点, 满贯以上听张数.
- 骗现: 听立直家/副露家现物.

## 鸣牌策略

- 鸣牌是赤牌麻将中非常重要的技术.
- 鸣牌优点: **速度上升**.
- 鸣牌的缺点: **打点下降**, **防御力下降**.

### 鸣牌选择

- **两张以上宝牌**时候要重视速度与和牌的容易度,
  把鸣牌作为做牌的前提.
- 亲家速攻: 鸣牌提升和牌速度, 保持连庄.
- 可以考虑鸣牌:
  - 满贯手牌: e.g 多宝牌/染手.
  - 消除愚型 (嵌张吃, 边张吃, 绝张吃/碰): 愚型过多, **门清立直速度过慢**.
  - 两副露: 良型一向听, 良型两向听, 打点 3900 以上 (防止过度鸣牌).
  - 三副露: 良型听牌, 一向听满贯手役 (防止过度鸣牌).
  - 晚巡低向听 (急于听牌).
  - 破除一发.
  - 速攻阻止对手大牌.
- 不建议鸣牌:
  - 牌型差, 打点低. 鸣牌对于速度增益小, 不鸣牌可保持高打点和防御力.
  - 向听数不变且没有改良.
  - 安全张少 + 高向听: 不鸣牌可以有一对安全张.
  - 鸣牌**过早**或鸣牌**过多**都会导致他家提高警惕性, 使得和牌可能性变小.

### 吃牌策略

- 吃牌时尽量选择嵌张/边张吃, 进张数最大化.

### 碰牌策略

役牌为**唯一雀头**时, 若牌型较好(一向听), 且立直门清打点显著高于役牌打点, 则不急于鸣役牌:

- 唯一雀头鸣牌不能降低向听数.
- 胡一番役牌意义不大, 副露降低防守能力.

### 杠牌策略

- 幺九牌暗杠 32 符, 相当于 1 番.
- 杠宝牌/岭上开花增强手牌番数.
- 杠牌增加一次摸切机会.
- 不建议杠牌:
  - 牌型差, 多愚型, 高向听.
  - 进张减少 (甚至退向): e.g 中张暗刻不开杠, 保留作搭子 (暗刻复合型).
  - 他人立直或副露, 自己和牌率较低.
  - 手牌打点足够, 无需杠宝牌.

## 手役策略

### 常见手役

- 一番役:
  - 立直: **(门清)** 报听.
  - 役牌: `场风牌`/`自风牌`/`三元牌` 刻子或杠子.
  - 平和: **(门清)** 四顺子 + 两面听 + 非役牌雀头.
  - 断幺九: 二八数牌.
  - 一杯口: **(门清)** 两相同顺子.
- 二番役:
  - 三色同顺: **(降番)** 三组数字相同, 花色不同的顺子.
  - 七对子 (七对): **(门清)** 不可有相同对子.
  - 对对和 (对对). 四面子都是刻子或杠子.
  - 一气贯通 (一通): **(降番)** 1-9 同色顺子.
  - 三暗刻: 手牌有三暗刻或暗杠.
- 三番役:
  - 混一色 (混一): **(降番)** 所有元素由同一种花色和字牌构成.
- 六番役:
  - 清一色 (清一): **(降番)** 所有元素由同一种花色构成.

- 配合自己的摸牌来作手役是一个准则, 禁止刻意作大牌 (强行拆掉多个搭子).
- `有手役` = `能够鸣牌` = `速度快`, 手役用来提高速度是现代麻将的一种思想.
- 打点 (番数) 相同时, 确定的手役比不确定的手役更有利:
  - 断幺九接近成型, 可以选择切掉役牌对子 (未成型), 保留其他中张的对子或搭子, 选择断幺九手役.
  - 断幺九并未成型, 可以选择保留役牌对子, 切掉其他超载的对子或搭子, 选择役牌手役.
- 不损失或较小损失牌效时, 应在**充分考虑高打点手役**的前提下进行切牌:
  - 宝牌: **早巡**保留宝牌及其靠张.
  - 平和手役: 听好型.
  - 断幺九手役: 切幺九牌向内靠拢.
  - 三色同顺手役: 切牌考虑三色可能.
  - 一气贯通手役: 保留多牌混合型.
  - 染手手役: 切牌考虑染手可能, 同色牌较多时积极鸣牌.
- 领先时, 速攻 (役牌, 早巡立直) 抢庄是保持一位的有效手段.
- 低打点时, 不应盲目和亲家对攻 (放铳代价很大).
- 混全带, 纯全带, 混老头等手役在赤牌麻雀中收益不高, 且防御力很弱 (需要切中张).
- 手役会限制牌形 (防守 ↓), **作手役追求打点**切牌大概率不是进张数最大化的切牌 (速度 ↓), 是其不可忽视的两大缺点.

### 立直手役

- 立直可视为 1.5 番.
- 立直有威吓力, 迫使对手防守, 破坏对手和牌, 增加自摸可能性.

#### 默听选择

- 立直和默听打点界限为**满贯**.
- 满贯/跳满以上, 一般选择默听.
- AL + TOP + 确定手役, 选择默听.
- **低打点听牌**时, 默听可以有**弃和**机会:
- **愚型听牌**时, 默听可以有**改良**机会:
  - 子家早巡愚型听牌, 当改良枚数是和牌枚数两倍以上时, 可以暂时默听寻求改良.
  - 亲家早巡愚型听牌, 可以考虑立即立直.
- **愚型默听**和牌率 ~= **良型立直**和牌率.

#### 立直选择

- `无役`有`宝牌`和`赤牌`的愚型听牌立直是上等选择.
- 良型听牌应立即立直.
- 四暗刻立直, 逼迫对手防守, 延长巡目, 提高自摸可能性.
- 先制立直时, 愚型听牌高于良型听牌两番时, 可以追求高打点, 放弃听张数.
- 追立时, 应优先听张数, 一定程度上放弃高打点.
- 平场或亲家时, 良型低打点听牌立直 > 愚型高打点听牌立直 > 默听.

#### 门清选择

- 亲家愚型搭子过多, 且有确定手役 (役牌/断幺九) 时,
  应**积极鸣牌**, **放弃门清**, 加速和牌.

### 平和手役

- 作两面是作平和手役的要点.
- 切掉役牌雀头, 保持断幺平和的可能.

### 断幺九手役

- 断幺九**可鸣牌 + 赤宝牌**可以做到速攻多番.
- 早中巡**二向听**手牌, 通常是**鸣牌断幺九** (鸣牌后切幺九牌) 的好开始点:
  - **一向听**手牌可以追求门清.
  - **高向听无宝牌**恶型手牌不急于鸣牌: 鸣牌对于高向听帮助不大, 反而降低打点与防御力.
- **多张宝牌**: 和牌速度至关重要, **鸣牌断幺九**可以加速和牌.
- **亲家**愚型搭子过多时, **愚型搭吃牌** + **断幺九**可以加速和牌.
- 断幺九容易作成, 容易与其他役复合, 摸切过程中需要**时刻考虑断幺九**可能性.

### 役牌手役

- 好型搭子超载时, 优先切掉单张役牌.
- 愚型搭子超载时, 可以保留单张役牌, 等待役牌对子, 寻求快速听牌.

### 宝牌策略

- 宝牌: **早巡**保留宝牌及其靠张.
- 宝牌靠张 (特别是愚型靠张) 会降低和牌速度, 降低手牌防御力, 存在提升打点可能性:
  TOP 或速攻时应及时切掉, 落后或低打点时可适当保留.
- **多张**宝牌: 好型足够时可考虑门清立直, 一般情况下积极鸣牌速攻.

### 染手手役

- 具有`立直`(**多好型**)/`断幺`/`三色`/`一气`等其他可能手役时, 不要强行染手.
- 中巡后染手手役未成型, 转牌效切牌, 不要强行染手.
- 可以考虑染手手役:
  - 10 张以上字牌 + 同色牌.
  - 2 对以上字牌 + 4 张以上同色牌:
    - 等价于 8 张以上染手有效牌.
    - 字牌鸣牌**容易**, 染手手役 + 双役牌手役可达成**满贯**.
- 牌型较差的混一色手役要**重视防守**:
  - 高向听时不要轻易鸣牌.
  - 优先切中张给对手读牌压力, 保留**字牌**作为后续安全牌 (他人立直或听牌).
    (染手手役中的字牌在弃和后可用于防守).
- 牌型较好的混一色手役切牌时适当隐藏染手倾向 (e.g 优先切边张), 降低字牌鸣牌难度.

### 三色同顺手役

- 同等牌效切牌时, **时刻注意三色同顺**手役的自然形成.
- 当已有**两组**非三色**成型面子**时, 三色同顺手役可能性大幅下降 (需要三组面子).

### 一气贯通手役

- **六枚**同花色为一气贯通高概率状态.

### 对对和手役

- 幺九牌容易鸣牌, 中张数牌不容易鸣牌.
- 可以考虑对对和手役:
  - 早巡多对幺九对子 + 3900 打点.
  - 至少一组 `刻子` + 多组 `对子` (尽可能保留**幺九对子**).
- 早巡 3 对子 + 0/1 面子 + 多愚型, 可以考虑对子手役 (**普通面子手役速度过慢**).

### 七对子手役

- 四对子: 牌型与进张较差时, 可考虑七对子手役.
- 五对子: 早巡七对子一向听, 可考虑转对对和手役.
- 六对子: 中晚巡七对子听牌, 禁止转对对和手役.
- 亲家作七对子优先考虑和牌率 (保留幺九对子), 子家作七对子可考虑打点 (断幺九 + 七对子).

### 全带幺手役

- 五组以上幺九搭子 + 对子时可以考虑全带幺手役, 优先切中张牌.

## 振听规则

处在振听状态下, 无法荣和, 只能自摸.

### 捨牌振听

- 曾打出过当前听牌 (胡牌型, 不考虑手役), 进入捨牌振听, 所有听牌都无法荣和, 只能自摸.
- 解除方法: 换听其他没有振听的牌.
- 意味着可以**安全打出**其他三家**河牌与副露**,
  通过看牌避免放铳, 强调立直麻将的防守性.

### 同巡振听

- 听牌时, 没有荣和他人的牌, 进入同巡振听.
- 解除方法: 下一次自己打出任何牌.
- 意味着在一巡中可以**安全打出**上家打出过的牌,
  通过看牌避免放铳, 强调立直麻将的防守性.

### 立直振听

- 立直听牌时, 若出现可以胡的牌, 但没有进行胡牌, 进入立直后振听.
- 一直持续到此局结束, 意味着此局只能自摸.

:::tip 竞技麻将
振听规则是奠定立直麻将防守战术的基石.
:::

## 防守策略

### 安全牌

- 三家现物.
- 熟张字牌.
- 完全 NO CHANCE 牌.
- 留安: 在搭子足够的前提下, 损失一定牌效 (一般为 2 张左右), 保留`熟张字牌`/`NC 边张`/高威胁对手的`现物`作为安全牌.
  - 切安全牌进行立直, 稳健进攻.
  - 切安全牌进行兜牌, 不损牌效.
  - 防止一发.
  - 保留安全牌的同时, 可以优先切掉高威胁对手的`可能听牌` (**危险张**), 保证自己**听牌时不放铳**.

### 壁牌

- 壁牌: 四枚可见, 对应 NO CHANCE 牌.
- 薄壁牌: 三枚可见, 对应 ONE CHANCE 牌.
- 壁牌/薄壁牌外侧牌相对安全, 内侧牌仍有放铳可能.

### 筋牌

表筋 `147`, `258`, `369`:

- 满筋 (**安全**): 现物有 `1` `7`, 则 `4` 是满筋牌.
- 边筋 (**安全**): 现物有 `4` `6`, 则 `1` `9` 是边筋牌.
- 半筋: 现物只有 `1`, 则 `4` 是半筋牌, 不太安全 (e.g `156` 切 `1` 听 `47`).

### 早巡外侧牌

早巡手切边张, 外侧牌安全.

### 弃和

安全度:

- 安全牌: 现物 > 熟字.
- 壁牌: NO CHANCE > Double ONE CHANCE > ONE CHANCE .
- 筋牌: 边筋 `19` >> 边筋 `28` = 满筋 `456` > 筋牌 `37`
- 早外牌: 早巡 `2378` 外侧牌放铳率降低 30%.
- ONE CHANCE 危险度: (筋牌, 无筋牌).
  中晚巡, ONE CHANCE 危险度急剧上升.
- 边张 > 生张 > 宝牌生张.
- 立直宣言牌的所在筋牌存在**骗筋听牌**可能, 危险度视为无筋牌.
- 立直宣言牌的另外两条筋牌存在**好型听牌**可能, 危险度上升.
- 染手手役 (混一色/清一色) 没有筋牌防守理论, 同色牌都是**危险张**.
- 对子手役 (对对和/七对子) 没有筋牌防守理论, `NO CHANCE`/`役牌`/`生张`都是**超危险张**.
- 鸣牌后切出的同色牌危险度上升.

#### 弃和选择

- 某一家牌河有高打点迹象, 另外两家仍直线进攻, 自家手牌高向听, 应切掉危险张留好安全牌, 随时作弃和准备.
- 他人立直或多副露听高打点牌, 且自己和牌困难 (e.g `23456` 向听/愚型搭子多且进张少) 时,
  除非情况特殊, 应完全弃和.

#### 弃和策略

- 未有人听牌 (早巡) 时, 优先打危险张; 有人听牌后, 顺序打安全张.
- 尽量持有多家共同安全张, 优先打立直选手特有的现物.
- 不要过度依赖筋牌, 要结合壁牌/熟张/场况进行判断.
- 跟打策略 (同巡振听): 优先**跟着上家**出牌, 在对手直接摸切时才可跟着对家或下家出牌.
- **配牌弃和**:
  配牌极差, 没有和牌可能.
  可以**优先切中张**, **保留幺九牌**,
  即可切掉危险牌保留安全牌,
  又可制造染手/国士无双等大牌假象.

### 兜牌

保留危险张的同时, 向听牌前进:

- 拆对子.
- 拆搭子.
- 拆暗刻.
- 七对子: 最强防守牌型, 又有和牌可能.

### 副露读牌

- 听牌可能:
  - 三副露.
  - 二副露 + 若干巡手切改良.
  - 晚巡一副露.
  - 染手副露 + 切同色牌.
  - 断幺九副露 + 切中张宝牌.
  - 对对和副露 + 切中张好型.
  - 小三元副露.
- 连续摸切: 低向听.
- 宝牌副露: 防止放铳, 不应盲目对和.
- 断幺九副露: 不应直接弃和.
- 副露 + 切好型 + 持续摸切**大概率听牌**.

6-10 巡听牌率:

- 一副露: 15%.
- 二副露: 34%.
- 三副露: 71%.

13 巡后听牌率:

- 一副露: 53%.
- 二副露: 70%.
- 三副露: 87%.

### 牌河读牌

里筋: 靠近内侧的筋.

- X < 5, X 里筋 = `X+1` 和 `X+4`.
- X = 5, X 里筋 = `X-4`, `X-1`, `X+1`, `X+4`.
- X > 5, X 里筋 = `X-1` 和 `X-4`.
- 里筋自身互为筋牌 (差 3).

跨筋/靠张/周边:

- `X-2`.
- `X-1`.
- `X+1`.
- `X+2`.

牌河读牌策略:

- 跨筋: **晚巡**手切 `5`, `5` 跨筋危险度提升.
- 里筋: **早巡**手切 `5`, `5` 里筋危险度提升.
- 双里筋 (间四间防守):
  **早巡**手切 A 和 B, `A + 5 = B`, A 和 B 的里筋 (**相同**) 非常危险.
- 进攻信号: 乱序切牌, 切宝牌及其周边, 先切安全牌后切危险牌, 早巡多次跟打字牌或生张役牌.
- 防守信号: 跟打现物, 切面子, 先切危险牌后切安全牌.
- 早巡多次摸切: 超好型或进张极差.
- 直线乱序切牌: 高打点兼顾防守 (保留边张与字牌作为安全牌); 顺序切牌后乱序切牌: 高打点放弃防守.
- 亲家副露, 同巡子家切危险中张: 子家高打点可能.
- 切掉一组搭子 (e.g 连续手切 `75`): 搭子超载, 手牌速度较快.
- 切掉一组好型搭 (e.g `56` 两面搭): 好型搭超载, 手牌速度较快,
  听两面好型概率很大, 可以利用**筋壁理论**进行防守.
- 切掉**唯一**役牌对子: 选择断幺九手役, 手牌速度较快.
- MPS 法则: 切掉三枚同数牌 (e.g 6m6p6s), 手牌速度较快.
- 切宝牌存在两种可能：
  - 满贯以上高打点 (放弃宝牌).
  - 好型一/二向听 (孤张宝牌).
- 牌河宝牌个数: 推测对手手牌打点.
- 立直宣言牌:
  一般情况下, 切 `5` 立直两面听 `36` 或 `47`,
  当牌河出现 `19` 时, 说明存在嵌张听牌骗筋可能, 立直宣言牌的筋牌 `2` 与 `8` 危险度上升.

## 场况策略

TOP 时策略 (保守策略):

- 不盲目对攻, 不盲目立直, 尽量默听.
- 避免开杠.
- 保留安全牌.
- 尽快和牌, 不追求大牌.
  - 速攻结束牌局.
  - 速攻抢庄.

四位时策略 (激进策略):

- 积极进攻.
- 不保留安全牌.

亲家策略:

- 速攻连庄.
- 对攻积极.
- 优先保证和牌率 (一定程度上放弃打点).
- 亲家进攻威慑力大 (立直 > 默听).

子家策略:

- 立直追求满贯, 副露追求宝牌或染手.

早巡策略:

- 当牌型较好时, 可以损失一定牌效去追求打点 (断幺/三色/染手).

### 攻守选择

基本判断策略:

- 四家点数: 落后激进, 领先保守.
- 和牌容易度.
- 和牌打点.
- 放铳危险度 (对手打点).
- 弃和成功率.

> 基本对攻策略 (对手立直或三副露): 听牌攻, 一向听攻一巡, 高向听弃和.

良型听牌进攻策略:

- 子家:
  - 8 巡: 手牌点数 > `对手 * 1/3`.
  - 12 巡: 手牌点数 > `对手 * 1/2`.
- 庄家
  - 8 巡: 全攻.
  - 14 巡后: 两番.

愚型听牌进攻策略:

- 子家:
  - 6 巡: 手牌点数 > `对手 * 2/3`.
  - 12 巡: 手牌点数 ~= 对手.
- 庄家:
  - 6 巡: 全攻.
  - 8-15 巡: 两番.
  - 15 巡后: 三番.

> 他人立直或听牌后, 自己处于 `23456` 向听, 除非情况特殊, 应完全弃和.

### 点数计算

|      | 中张牌 | 幺九牌 |
| ---- | ------ | ------ |
| 明刻 | 2 符   | 4 符   |
| 暗刻 | 4 符   | 8 符   |
| 明杠 | 8 符   | 16 符  |
| 暗杠 | 16 符  | 32 符  |

- 基本符数: 20 符.
- 役牌雀头/嵌张听/边张听/单骑听 2 符, 自摸 2 符, 门清荣和 10 符.
- 七对子 25 符, 平和自摸 20 符, 平和荣和 30 符, 自摸/鸣牌荣和至少 30 符, 门清荣和至少 40 符.

| 番数          | 称呼   | 基本点                  |
| ------------- | ------ | ----------------------- |
| < 2000        |        | `2 ^ (番数 + 2) * 符数` |
| 五番以下      | 满贯   | 2000                    |
| 六 - 七番     | 跳满   | 3000                    |
| 八 - 十番     | 倍满   | 4000                    |
| 十一 - 十二番 | 三倍满 | 6000                    |
| 十三番以上    | 役满   | 8000                    |

|      | 荣和         | 自摸                                     |
| ---- | ------------ | ---------------------------------------- |
| 子家 | `基本点 * 4` | 亲家赔`基本点 * 2`, 子家各赔`基本点 * 1` |
| 亲家 | `基本点 * 6` | 子家各赔`基本点 * 2`                     |
